# Dostaffkin - Сервис доставки посылок

**Dostaffkin** — веб-приложение для заказа и отслеживания доставки посылок между странами и внутри стран. Интеграция с Яндекс.Картами для построения маршрутов и расчёта стоимости.

---

## Функциональность

- **Расчёт стоимости доставки**
  - Выбор пунктов отправления и назначения (подсказки адресов от Яндекс.Карт)
  - Выбор размера посылки (6 тарифов: от XS до MAX)
  - Выбор скорости доставки (обычная / приоритетная)
  - Построение маршрута на карте, расчёт расстояния, длительности и итоговой цены

- **Оформление заявки**
  - Ввод контактных данных (имя, телефон, комментарий)
  - Отправка данных на бэкенд
  - Получение номера отправления

- **Отслеживание посылки**
  - Поиск по номеру отправления
  - Отображение маршрута и статусов доставки (создан, в пути, готов, выполнен)

---

## Технологический стек

- **Frontend:** Angular 17 (standalone components, signals, reactive forms)
- **Карты:** Яндекс.Карты API (геокодер, мультимаршрутизация, подсказки)
- **HTTP-запросы:** Angular HttpClient
- **Стили:** CSS (адаптивная вёрстка, кастомные шрифты Roboto)
- **Бэкенд (внешний):** `https://testologia.ru/delivery`

---

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm start
```

Приложение будет доступно по адресу `http://localhost:4200`.
